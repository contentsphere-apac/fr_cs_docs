Multi-classify Objects¶

The Multi-classify Objects feature provides general users the ability to add attribution classes to the product in addition to the single Nature Class.

Description:
  • To multi-classify the product, we add multiple attribution classes to the product in addition to the single Nature Class. Attribution classes contain detailed descriptive properties for the product.
  • The user is able to define the product further by adding properties based on different attribution classes.
Related topics:
  • Attribution classes with attributes and tags
Application:

  • Attribution classes, which are also known as Non-nature classes, contain more detailed information about the products. The nature class would only have general properties.

  • The attribution class would define the product more in detail.

  • Example:

    For an energy drink, the nature class Single Article would just contain generic properties for the product, whereas an attribution class Beverages would give details regarding beverages such as volume and can size. As for another class Energy Drinks, it would give properties such as nutrition contents and caffeine contents.

How to do: Multi-classify Objects¶

This How To guide provides instructions on how to multi-classify objects. To do this, follow these steps:


  1. Go to PIM and hover on the “+” icon.


  1. Click Single Article.


  1. Enter the name of the single article.


  1. To add a Non-Nature class or Attribution class, click Add Class.


  1. Select a class from the drop-down list.



  1. To add another attribution class, click Add Class and select from the list.



  1. To switch templates from nature class to non-nature class, click on the drop-down list and select an option, for example Beverages.
